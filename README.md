# Selenium UI Automation Tests

## Overview
This project demonstrates automated UI testing using Selenium WebDriver with Java and JUnit.  
The tests simulate user interactions on the W3Schools website and verify that pages and elements load correctly.

## Test Cases
- Search for **"Java tutorial"** and verify the tutorial page loads
- Navigate to **CSS Practice Tests** and verify page content

## Tools Used
- Selenium WebDriver
- Java
- JUnit
- ChromeDriver
